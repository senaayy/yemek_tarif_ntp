## 3. EKRAN GÖRÜNTÜSÜ ALMA TALİMATLARI

### 3.1 Windows'ta Ekran Görüntüsü Alma

1. **Snipping Tool Kullanımı:**
   - Windows tuşu + Shift + S
   - İstediğiniz alanı seçin
   - Görüntüyü kaydedin

2. **Print Screen Kullanımı:**
   - Print Screen tuşuna basın
   - Paint veya başka bir görüntü düzenleme programında yapıştırın
   - Kaydedin

### 3.2 Önerilen Ekran Görüntüleri

- Ana ekran (boş form)
- Malzeme girişi yapılmış ekran
- Tarif önerisi sonuç ekranı
- Besin değerleri gösterilen ekran
- Favori listesi ekranı
- Favori detay ekranı
- Hata mesajı ekranları

## 4. DEMO VİDEOSU ÇEKME TALİMATLARI

### 4.1 Video Kayıt Araçları

- **OBS Studio:** Ücretsiz, profesyonel ekran kayıt yazılımı
